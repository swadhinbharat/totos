# totos
A utility program to do simple repetetive tasks

## Installation

```bash
npm install -g totos
```

## Options
- [Version](#version)
- [IP](#ip)

### Version
```bash
# Get version
totos -v
totos --version
```


### IP
```bash
# Get all IP
totos ip

# Get all Local
totos ip -l
totos ip -local

# Get all Global
totos ip -g
totos ip -global
```
